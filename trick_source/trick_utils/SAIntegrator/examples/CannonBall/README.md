# CannonBall

This is an example of using the RK2Integrator to create a simple cannon ball simulation.

```
$ make
$ ./CannonBall > cannon.csv
$ python plot_trajectory.py
```

![Cannon](images/Cannon.png)

```
$ make
$ ./Cannonball
```
